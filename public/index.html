Connection Guide

OBS Studio Setup

  1. Open OBS → Settings → Stream
  2. Service: Custom...
  3. Server: rtmp://your-domain:1935/live
  4. Stream Key: Copy from stream settings
  5. Click Start Streaming

FFmpeg Command

ffmpeg -re -i input.mp4 -c:v libx264 -c:a aac \ -f flv rtmp://your-domain:1935/live/STREAM_KEY
